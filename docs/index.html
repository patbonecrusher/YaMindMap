Why YaMindMap?

⚡

GPU-Accelerated

Native rendering via wgpu — Metal on macOS, Vulkan on Linux, DX12 on Windows. Smooth 60fps with 1000+ nodes.

🎨

Drag & Drop

Reparent and reorder nodes by dragging. Visual bezier connectors show where nodes will land. Full undo/redo.

🛠

Balanced Layouts

Automatic left/right balanced map layout, tree layout, and more. Nodes are positioned intelligently as you build.

🌐

Truly Cross-Platform

One codebase, native on macOS (ARM), Windows (x64/ARM), and Linux (x64/ARM). No Electron, no webview.

💾

Open Format

Documents saved as readable JSON. Export to PNG, SVG, and PDF. Your data is always yours.

⌨

Keyboard-First

Tab for child, Enter for sibling, Delete to remove. Navigate with arrows. Every action has a shortcut.

Built with Modern Tech

No compromises on performance or portability.

Rust iced wgpu Metal Vulkan DX12 R-tree spatial index Bezier edge routing Command pattern undo/redo
